# Usage
<!-- usage -->
```sh-session
$ npm install -g bunder
$ bunder COMMAND
running command...
$ bunder (-v|--version|version)
bunder/0.0.0 linux-x64 node-v10.16.0
$ bunder --help [COMMAND]
USAGE
  $ bunder COMMAND
...
```
